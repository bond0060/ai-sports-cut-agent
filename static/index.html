篮球投篮检测

上传视频后自动根据篮筐位置生成 ROI，按固定比例规范检测区域，提升不同测试视频的可比性。

拖拽视频到此处

或点击选择文件

支持 MP4 / MOV / AVI / MKV / WEBM / M4V

选择观察区域 (ROI)

拖动黄框移动位置，拖角点调整大小。尽量只覆盖篮筐及上方入球空间，避开球员、球架等干扰。

有网时：进球需同时满足「轨迹穿筐」+「篮网形变」；可过滤暗光下人头误检。无网场地请关闭此项。
